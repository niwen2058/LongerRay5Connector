To use this app, run ray5_connector.py or compiled exe from dist folder. Then you'll need to pass IP address of a laser.
Software will remember IP and MAC address of a laser, and will try to reconnect automatically after next run - first over known IP, later by scanning network for known MAC.
Features include:
- Connect, disconnect, automatic reconnect after program start
- Keep alive green dot, marking proper connection
- SD card fil list
- multiple files selection for adding and deleting
- Drag and drop multiple file transfer
- Green fading background for new files

Software was generated using Gemmini 3 Flash, development with RE took 87min. 
It can be used free of charge for any commercial/non commercial work, in part or fully.
